# Porfolio Personal

Este repositorio contiene mi porfolio personal, una aplicación web pensada para presentar mi perfil profesional, experiencia, stack tecnológico, proyectos destacados, certificaciones y formas de contacto en una sola experiencia visual.

Además de funcionar como sitio de presentación, este proyecto también sirve como base para mostrar el tipo de interfaces y experiencias que desarrollo.

View your app in AI Studio: https://ai.studio/apps/a0551abe-067d-4165-ab5c-dac738e09c96

## Sobre el proyecto

Este porfolio fue desarrollado como una SPA moderna con foco en diseño, navegación fluida y estructura clara de contenido. Incluye secciones como:

- Hero principal de presentación
- Sobre mí
- Tech stack y habilidades
- Proyectos destacados
- Certificaciones
- Experiencia
- Contacto

## Tecnologías utilizadas

- React
- TypeScript
- Vite
- Tailwind CSS
- Motion
- Node.js / Express

## Run Locally

**Prerequisites:**  Node.js

This contains everything you need to run your app locally.

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

## Objetivo

El objetivo de este proyecto es mostrar de forma clara y visual mi trabajo, mis capacidades técnicas y algunos de los proyectos que mejor representan mi perfil como desarrollador.
